# AI Output Evaluation Harness
 
A test harness for evaluating LLM output quality — checking for hallucination, consistency, and formatting drift across non-deterministic AI pipeline outputs.
 
## Status
🚧 In progress
 
## Goal
Run a structured set of test prompts against an LLM pipeline and score the outputs for reliability, using an LLM-as-judge approach instead of exact-match assertions.
 
## Tech Stack
- Java
- JUnit 5
- LangChain4j
